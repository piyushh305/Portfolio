# 🚀 Piyush Yadav | Personal Portfolio

A sleek, modern, dark-themed personal portfolio website built to showcase my background in AI & Data Science, academic achievements, and technical projects. Designed from scratch with a focus on smooth animations, glassmorphism, and responsive UI.

🌐 **Live Demo:** [Insert Link Here]

## ✨ Features

* **Modern Dark UI:** A deep black/dark-grey aesthetic (`#000000` to `#0a0a0a`) with vibrant, high-contrast gradient accents (Blue to Green to Orange).
* **Smooth Animations:** Custom CSS keyframes for floating orbs, pulse effects, and seamless staggered fade-ins triggered by an `IntersectionObserver`.
* **3D Card Tilt Effect:** Interactive project cards that tilt dynamically based on mouse position using Vanilla JavaScript.
* **Glassmorphism Navigation:** A sticky navbar with dynamic background blurring (`backdrop-filter`) that reacts to scroll depth.
* **Interactive Modal:** A custom "Hire Me" modal overlay for quick contact access.
* **Fully Responsive:** Adapts flawlessly to mobile, tablet, and desktop screens using CSS Grid and Flexbox.
* **Zero Dependencies:** Built entirely with raw HTML, CSS, and Vanilla JavaScript. No heavy frameworks or libraries.

## 🛠️ Tech Stack

* **Markup:** HTML5
* **Styling:** CSS3 (Custom Variables, Flexbox, CSS Grid, Media Queries, CSS Transitions & Keyframes)
* **Interactivity:** Vanilla JavaScript (DOM Manipulation, Event Listeners, IntersectionObserver API, Math for 3D calculations)
* **Typography:** [Syne](https://fonts.google.com/specimen/Syne) for bold, modern headings and [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) for clean, tech-focused data points.
